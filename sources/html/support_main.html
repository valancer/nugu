NUGU앱은 T아이디로 로그인이 가능합니다. T아이디는 SK텔레콤의 서비스를 이용할 수 있는 통합 계쩡입니다. T아이디가 없으시거나, 기억이 나지 않으세요? 지금, 아래 탭 중에서 해당하는 것을 클릭해주세요
  • T월드 아이디가 있어요

    T월드 아이디가 있어요

    T월드 ID가 있으시면,약관 동의를 통해 T아이디로 전환해서 사용하실 수 있습니다

    1. T월드 ID로 로그인
    2. T아이디 약관동의
    3. T아이디 전환 완료
  • T월드 아이디가 없어요

    T월드 아이디가 없어요

    'T아이디 가입하기'를 선택하시면, 통신사에 관계없이 신규 가입을 하실 수 있습니다.

    1. T아이디 가입하기 선택
  • 아이디가 기억이 안나요

    아이디가 기억이 안나요

    본인 인증을 통해, T아이디를 찾아보세요

    1. 아이디 찾기 선택
    2. 본인 인증
    3. ID 찾기 완료
  • 아이디를 도저히 못찾겠어요 ㅠ.ㅜ

    아이디를 도저히 못찾겠어요 ㅠ.ㅜ

    아이디를 아무리 찾아도 모르겠으면, 본인 인증 없이 신규 가입을 진행해보세요

    1. T아이디 가입하기 선택
    2. 아니오 선택
    3. 약관 동의
    4. ID/비밀번호/개인 정보 입력
